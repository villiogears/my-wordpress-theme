# my-wordpress-theme

このリポジトリはシンプルな WordPress テーマの土台です。

## インストール

1. このフォルダ全体を WordPress の `wp-content/themes/` フォルダにコピーします（例: `/path/to/wordpress/wp-content/themes/my-wordpress-theme`）。
2. WordPress 管理画面の「外観 → テーマ」から「My WordPress Theme」を有効化します。
3. 外観 → メニューでメニューを作成し、Primary Menu に割り当ててください。
4. ウィジェット（外観 → ウィジェット）でサイドバーにウィジェットを追加できます。

## 特徴

- レスポンシブなシンプルレイアウト
- タイトルタグ自動管理
- アイキャッチ（サムネイル）対応
- カスタムメニューとウィジェット領域

## カスタマイズ
テンプレートファイル（`header.php`, `footer.php`, `index.php`, `single.php`, `page.php`, `sidebar.php`）を編集して自由にカスタマイズしてください。
